## Project description
Fullstack car rental web applicaton, designed and built with React, Node.js and Ant Design.

## Installation
**Install server dependencies**
```bash
npm install 

```

**Install client dependencies**
```bash
cd client
npm install 
```

**Run development server**
```bash
npm run dev
```
